【第1章】 基本設定とUIアイコンChapter 1: Basic Setup & UI

⌨️

キー割り当てのルール (Your Keys / Original Keys) 🆓

Key Binding Rules 🆓

対象サイトの「✏️ 編集」を押し、あなたが実際に使いたい送信キー・改行キーを設定します。これだけで基本的なショートカットの統一が完了します。

Edit the target site and set your preferred send/newline keys to unify your shortcuts.

👉 詳しい設定項目を見る👉 View Detailed Setup
  • ① 送信キー (自分): あなたが実際に使いたい理想の送信キー（例: Shift+Enter）。
  • ② 改行キー (自分): あなたが改行に使いたいキー（例: Enterのみ）。
  • ③ 送信 / ④ 改行キー (元々): そのサイトが元々標準で採用しているキー。当アプリがこの本来のキー信号を偽装してサイトに送るために、必ず正しく設定しておく必要があります。
  • ① Send Key (You): Your ideal send shortcut (e.g. Shift+Enter).
  • ② Newline Key (You): Your ideal newline shortcut (e.g. Enter).
  • ③ Original Send / ④ Original Newline: The default keys originally used by the site. This is strictly required so the extension knows what signal to fake when bypassing the site.
🎯

自動取得 と スマートターゲット 🆓

Auto Select & Smart Target 🆓

プログラムが送信ボタンの場所（ボタンの住所＝CSSセレクタ）を特定するための強力な補助ツール群です。ご自身でコードを書く必要はありません。

Tools to automatically detect the CSS selector of the send button without coding.

👉 各アイコンの機能を見る👉 View Icon Functions
  • 🎯 自動取得 (Auto Select): このボタンを押したあと、実際のWebページの「送信ボタン」をクリックするだけで、アプリが自動でCSSセレクタ（ボタンの住所）を解析して入力してくれます。
  • ⚡ スマートターゲット: 送信ボタンのIDが毎回ランダムに変わってしまう厄介なサイト向けの機能です。「現在フォーカスしているテキストエリアの周辺」から、送信ボタンらしい要素をプログラムが推測して見つけます。
  • 🎯 Auto Select: Click this, then click the actual send button on the webpage. The extension will automatically parse and input the correct CSS selector.
  • ⚡ Smart Target: Useful for complex sites where button IDs change randomly. The program guesses and finds the send button based on elements near the currently focused text area.
📋

マルチCSSセレクタ (複数入力欄の対応) 🆓

Multi-CSS Selectors 🆓

ChatGPTやSlackのスレッドのように、1つの画面に「送信欄」が複数存在するサイトで使用します。動的検知をONにすることで、現在使っている入力欄のボタンを正確に狙い撃ちします。

For sites with multiple inputs on one screen (like Slack threads or ChatGPT). Enable dynamic detection to target the exact button for the input you are currently typing in.

送信後マクロ (自動タブ操作) 👑

After Send Macro (Auto Tab Action) 👑

送信が完了した直後に、ブラウザのタブを自動で操作する機能です。大量の問い合わせ処理や連続投稿など、ルーチンワークの効率を劇的に向上させます。

Automatically close or move to the next tab immediately after sending. Drastically improves efficiency for mass processing and continuous posting.

👉 操作の種類👉 Action Types
  • 現在のタブを閉じる: 送信完了と同時に、いま開いているタブを閉じます。
  • 隣のタブへ移動する: 送信後、右（または左）のタブへ自動でフォーカスを移します。「端のタブなら反対の端に戻る（ループ）」設定も可能です。
  • Close Current Tab: Closes the active tab immediately upon successful sending.
  • Move to Next Tab: Automatically moves focus to the right (or left) tab after sending. Can be set to "Loop to opposite end" if at the edge.

改行方式 (2種類) 🆓

Newline Methods (2 Types) 🆓

改行キーを押した際に、プログラムでどのように改行処理を行うかの設定です。

How the program handles inserting a newline when you press your chosen newline key.

👉 方式の違いを見る👉 View Methods
  • 信号偽装 (Signal): サイト側が用意している安全な「改行キーボード信号（Enter等）」を送ります。基本はこれを使います。
  • 強制挿入 (Insert): ブラウザの制御を無視し、拡張機能が現在のカーソル位置に直接「改行文字列（\n）」を書き込みます。
  • Signal Fake: Sends the safe native keyboard signal (like Enter) provided by the site. Generally the default choice.
  • Direct Insert: Bypasses browser control and writes the newline string (\n) directly at the cursor position.

送信カウントダウン機能 👑

Send Countdown Feature 👑

誤爆を防ぐため、送信ボタンを押した直後に指定した秒数（例: 3秒）のカウントダウンを開始します。宛先間違いに気づいたら「キャンセル」で止められます。

Start a countdown before sending. Click cancel if you notice a mistake.

送信確認ポップアップ 👑

Send Confirmation Popup 👑

送信の瞬間に画面中央に「本当に送信しますか？」という確認画面を出します。「Enter連打による誤爆」を防ぐため、操作方法を『クリックのみ許可』に限定することも可能です。

Show a confirmation popup. Can be restricted to mouse-click only to prevent rapid Enter key mistakes.

🚫

NGワード検知 🆓 / 👑

NG Words Filter 🆓 / 👑

登録した言葉が含まれている場合、送信時に検知して警告を出したり、送信を完全にブロックしたりできます。※「自動マスキング（***置換）」機能はUnlimited版限定です。

Blocks, warns, or masks when registered forbidden words are detected. (*Auto-masking is Unlimited only).

🪄

NGワード かんたん正規表現ビルダー 👑

NG Word Regex Builder 👑

難しいプログラム知識（正規表現）がなくても、電話番号やクレジットカード番号、メールアドレスを検知・伏せ字にするためのルールを画面上で直感的に作成できます。

Build complex regex for phone numbers or emails intuitively without coding.

👉 ビルダーの便利な機能👉 Builder Features
  • 定番プリセット: 「電話番号の下4桁だけ残して伏せ字にする」「URLをブロックする」など、よく使うルールをワンクリックで生成します。
  • 見たまま逆算: ***-****-####（#は残す数字、*は隠す数字）のように、最終的に表示させたい形を入力するだけで、賢くルールを逆算して作成します。
  • すり抜け対策: 全角/半角の違い、大文字/小文字、ひらがな/カタカナ、ローマ字入力（baka/vaka等）の表記揺れをすべて同一視する強力なオプションを備えています。
  • Presets: One-click generation for common rules like "mask phone numbers leaving last 4 digits" or "block URLs".
  • Reverse-Calc: Just type the final masked format you want (e.g. ***-****-####) and it smartly generates the rule backwards.
  • Anti-Evasion: Powerful options to treat full-width/half-width, upper/lower case, and various typographic variations identically to prevent bypassing.
🛑

時間帯送信ブロック 👑

Time-based Send Block 👑

指定した時間帯（例: 深夜や休日）は強制的に送信ボタンをロックし、意図しない時間外の業務連絡や誤爆を防止します。

Force lock the send button during specific hours (e.g. late night) to prevent off-hours communication.

画面上の便利なフローティングボタン 👑

Floating Widget Buttons 👑

Draggable floating widgets that you can place anywhere on the screen.

  • 📝 Counter: Real-time character count. Flashes red when you hit your defined limit.

拡張入力窓 (フローティングウィンドウ) 👑

Independent Floating Input Window 👑

元のサイトの入力欄が狭くて見づらい場合に、画面上に大きく独立した「拡張入力窓」を呼び出して快適に長文を作成できます。

Call up a large, independent input window to comfortably write long texts when the site's input box is too small.

👉 設定のポイント👉 Setup Points
  • 開閉キー: 任意のショートカットキー（例: Ctrl + M）で瞬時に窓を開け閉めできます。
  • ペースト時の改行処理: 「連続改行を半減」を選ぶと、リッチテキストからコピペした際に無駄に空いてしまう行間を自動で綺麗に整えます。
  • 送信キーの挙動: 窓の中で送信キーを押した時、「そのまま実際に送信まで行う」か「元のサイトの入力枠へ文字を移す（Apply）だけに留める」かを選べます。
  • Toggle Key: Open/close instantly with a shortcut (e.g. Ctrl + M).
  • Paste Formatting: "Halve newlines" automatically cleans up overly spaced paragraphs when pasting from rich-text documents.
  • Send Key Action: Decide whether pressing Send inside the window actually dispatches it, or merely "Applies" the text back to the main site's box.

Force Links to New Tab 👑

Force all links on the page to open in a new tab via a toggleable widget. Prevents losing your place during research.

Trigger Key & Templates 🆓

Press the Trigger Key (default Ctrl+Space) to open a template menu directly on the screen for instant pasting into the input box.

設定画面で「選択テキストをショートカットで保存」をONにし、キー（例: Ctrl+1）を割り当てておくと、ブラウザ上で文字を選択した状態でそのショートカットを押すだけで、一瞬で定型文としてアプリに保存できます。
Enable "Save selected text via shortcut" in settings. Highlight text in your browser, press the assigned shortcut (e.g. Ctrl+1), and it instantly saves as a template.

AI Integration & Tone Fixer 👑

入力窓の中にAIアイコン(✨)を追加し、あなたが書いた文章をAIが自動で「丁寧な言葉遣い」「簡潔な報告」などに書き直してくれます。

Adds an AI icon (✨) to the input window to automatically fix the tone of your text (e.g. make it polite or concise).

本機能はBYOK（Bring Your Own Key = 利用者自身のAPIキーを使用する）方式です。OpenAI、Anthropic、GeminiなどのAPIキーを直接設定画面に入力して使用します。

  • 対応プロバイダ: OpenAI (ChatGPT), Anthropic (Claude), Google Gemini, DeepSeek, Groq, OpenRouter, xAI など。設定画面の「🔑 取得ページ」から各社の発行画面へ飛べます。

This feature uses a Bring Your Own Key (BYOK) system. Enter your own API key directly in the settings.

  • Supported Providers: OpenAI, Anthropic, Gemini, DeepSeek, Groq, OpenRouter, xAI, etc.
  • Usage Costs: You are responsible for any API costs incurred from token usage.

Q. 送信ボタンが反応しない、改行がおかしくなります。Q. The send button doesn't react, or newlines break.
サイトによって内部の作りが異なります。「✏️ 編集」画面から以下をお試しください。
・送信されない時： 第2章「送信方式」を『CSSクリック』や『信号偽装』等に変える。
・改行文字が消える時（Teams等）： 第3章「システム文字入力方式」を『仮想ペースト』に変える。
・どうしてもダメな時： 「高度な方式を表示」にチェックを入れ、『[最強] React Fiber仮想DOM走破』などの特殊方式を試す。
Internal structures vary by site. Try editing the profile:
- Not sending: Change "Send Method" to CSS Click or Signal Fake.
- Text disappears (Teams): Change "Input Method" to Virtual Paste.
- Still failing: Enable advanced options and try ULTIMATE methods like React Fiber Traverse.
Q. Unlimited版（有料版）の機能は試用期間が終わるとどうなりますか？Q. What happens to Unlimited features after the trial?
インストールから30日間のフリートライアルが終了すると、自動的に無料版（制限あり）に切り替わります。設定画面の機能のうち、「👑」マークがついたウィジェット機能や自動化機能、および3つ目以降のカスタムプロファイルがロックされて使用できなくなります。勝手に課金されることはありません。
After 30 days, the extension automatically switches to the limited Free version. Features marked with 👑 and any Custom Profiles beyond the 2nd one will be locked. You will not be automatically charged.
Q. 突然すべてのカスタムプロファイルに「🔒」がついて使えなくなりました。Q. Custom profiles suddenly got a lock icon "🔒".
試用期間が終了し、かつご自身で追加したカスタムドメインが「3つ以上」登録されている場合に発生します。無料版ではカスタムドメインは最大2つまでしか有効化できません。設定画面上部の警告バナーから、「無料で使い続ける2つのドメイン」を選択していただくことでロックが解除されます。
This happens when your trial expires and you have more than 2 custom domains. The Free version allows a maximum of 2. Click the warning banner in settings to choose the 2 domains you want to keep.
Q. ライセンスキー（シリアルコード）を購入したのに認証でエラーになります。Q. License verification throws an error.
以下の点をご確認ください。
・入力ミス: 半角英数字ハイフンあり（ABCD-1234-EFGH-5678 の形式）で正しく入力されているか。
・通信エラー: 職場のネットワーク等で通信が遮断されていないか。「30秒経過」のメッセージが出た場合は、キャンセルせずに必ず同じシリアルキーのまま再送信を行ってください。
・認証上限: PCの買い替えや意図せぬアンインストール等による再認証は、1つのキーにつき「最大3回」までとなっています。上限を超えた場合は新規にご購入いただく必要があります。
Please check the following:
- Typos: Must include hyphens (Format: ABCD-1234-EFGH-5678).
- Network: Company firewalls might block it. If it times out, try sending it again.
- Limit Reached: Each key can be used/restored a maximum of 3 times due to PC changes or uninstalls.
Q. 会社のPCと自宅のPCなど、複数台で1つのライセンスを使えますか？Q. Can I use one license on multiple PCs?
はい。会社のPCとご自宅のPCで「同じGoogleアカウント（Chromeプロファイル）」でログインし、「拡張機能の同期」をONにしていれば、1つのライセンス認証状態が自動的に別のPCにも引き継がれます。同期を利用せず、手動で別々のPCに同じシリアルコードを入力して認証してしまうと「別のPC」としてカウントされ、再認証上限（3回）を無駄に消費してしまいますのでご注意ください。
Yes, if you use the same Google Account (Chrome Profile) and enable "Sync Extensions". The license state will sync automatically. If you manually enter the key on different PCs without sync, it consumes the limited restoration count.
Q. 拡張機能をアンインストール・再インストールしたらUnlimited版が消えました。Q. Reinstalling removed my Unlimited status.
セキュリティ上、拡張機能を一度アンインストールすると端末固有の内部IDがリセットされるため、未認証状態に戻ります。再度設定画面の「🔑」アイコンから、お持ちのシリアルコードを入力して再認証を行ってください（この際、復旧用として認証回数上限を1回消費します。トラブル時以外は安易にアンインストールしないでください）。
Uninstalling resets the internal device ID for security, removing your authentication. Re-enter your serial key via the "🔑" icon to restore it (this uses 1 of your 3 restoration counts).
Q. 入力窓（フローティングウィンドウ）が画面外に行ってしまい戻せません。Q. The floating window went off-screen.
マルチモニターの切り替えなどでウィンドウが画面外に消えてしまった場合、拡張機能の設定画面を開き、対象プロファイルの「🪟 拡張入力窓」の項目から一度チェックを外して保存してください。位置情報がリセットされ、次回有効化した際に画面中央に戻ります。
If the window is lost off-screen (e.g. after monitor changes), uncheck "Window ON" in settings and save. This resets its position to the center for the next time you enable it.
Q. 送信した直後に、勝手にタブが閉じて（移動して）しまいます。Q. The tab closes or moves automatically after sending.
プロファイル設定の「キー設定」内にある「🚀 送信後マクロ」が有効になっている可能性があります。設定画面を開き、対象サイトの送信後マクロの設定を「何もしない (基本)」に戻して保存してください。
Check your profile settings. The "🚀 After Send Macro" might be active. Change it back to "Do Nothing".
Q. 送信後マクロで「次のタブへ移動」がループしません。Q. "Move to next tab" macro doesn't loop.
「端のタブなら反対の端に戻る（ループ）」にチェックが入っていてもループしない場合、Chromeブラウザ自体がバックグラウンドタブの移動を制限している可能性があります。手動でタブをクリックしてアクティブにするか、ブラウザの再起動をお試しください。
Chrome may restrict background tab switching. Try clicking a tab manually to wake it up or restart the browser.
Q. NGワードを設定したのにすり抜けられてしまいます。Q. NG words are not blocked correctly.
相手が「ば か」のように間にスペースを入れたり、全角・半角を変えたりしている可能性があります。設定画面の「🪄 正規表現ビルダー」を開き、「詳細ビルド」タブから「すり抜け対策（全角半角の同一視、スペース許容など）」にチェックを入れて登録し直してみてください。
The user might be adding spaces (e.g. "b a d"). Open the Regex Builder, go to "Detailed Build", and enable Anti-Evasion options (allow spaces, ignore case) before saving.
Q. AI連携機能（トーン調整）でエラーが出て使えません。Q. AI integration throws an error.
以下の原因が考えられます。
1. APIキーの間違い: 登録したキーの文字列に余計な空白が含まれていないか確認してください。
2. クレジット残高不足: OpenAIやAnthropicのAPIはプリペイド式の従量課金です。各社のダッシュボードにログインし、支払い登録と残高（クレジット）があるかを確認してください。ChatGPT Plus等の月額プランとは別のアカウント管理になります。
3. モデル名のエラー: プロバイダの仕様変更によりモデル名が存在しない場合があります。「手動入力(カスタム)」から最新のモデル名を直接入力してお試しください。
Possible causes:
1. Key typo: Check for trailing spaces.
2. Insufficient Credit: OpenAI/Anthropic APIs are prepaid. Ensure you have credit in your dashboard (this is separate from Plus subscriptions).
3. Model Error: The provider may have changed model names. Try using "Custom Endpoint" and input the exact model.
Q. AIトーン調整を使うと元の文章が消えてしまいます。Q. AI tone adjustment deletes original text.
AIからの応答待ちの最中に、元の入力枠（または拡張入力窓）のテキストをユーザー自身が手動で書き換えたり消去したりすると、AIが書き換える対象を見失ってしまいます。AIの処理が終わるまではテキストエリアに触れないようにお願いします。
Do not manually edit or delete the text box while waiting for the AI response, as it will lose track of what to replace.
Q. 定型文（テンプレート）ショートカットが別の拡張機能と競合してしまいます。Q. Template shortcut conflicts with other extensions.
起動キー（デフォルトでは Ctrl + Space）がOSの言語切り替えや他のツールと被っている場合は、プロファイル設定の「⚡ 起動キー」項目から、別の組み合わせ（例：Alt + Shiftなど）に変更してください。
If the Trigger Key (default Ctrl+Space) conflicts with your OS language switch, change it to another combination like Alt+Shift in settings.
Q. 会社のPCで「仮想キーボード(.bat)」が実行できません。Q. Virtual Keyboard Host gives an error on company PC.
第7章の「仮想キーボード連携（.batの実行）」機能はWindowsのレジストリを書き換えるため、会社の厳しいセキュリティ設定やウイルス対策ソフトでは弾かれることがあります。
その場合は連携機能のご利用を諦め、「送信方式」から『仮想ペースト』や『React Fiber方式』などのソフトウェア完結の方式をお試しください。
The .bat file modifies the registry, which may be blocked by strict corporate security policies. Try software-based ULTIMATE methods like React Fiber Traverse instead.
